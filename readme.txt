== Description ==


Set up a Pokemon team through the settings page and allows you to display it wherever you want using the shortcode [pkmn_trainer].

**Features**
- Display between 1 &#8211; 6 pokemon

- Display a trainer sprite!


== Installation ==
1. Upload / Install plugin to plugin directory
2. Activate the plugin through the 'Plugins' menu in Wordpress.
3. Navigate to Settings > Pokemon Trainer Options.
4. Select Trainer, Pokemon
5. To display your team use the shortcode [pkmn_trainer] anywhere you want it to display
